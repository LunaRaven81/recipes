## Projektbeschreibung

Bei meinem Projekt handelt es sich um eine Website, auf der man diverse vegane Rezepte findet. Der Grund, eine eigene Website zu machen, war, da wir uns bis dato Rezepte in einer Instagram-Collection speicherten und Rezepte wiederzufinden dadurch sehr schwierig wurde. Und man auch teils mehrere Rezepte, welche das gleiche Gericht anders kochen, nicht nebeneinander fand.
